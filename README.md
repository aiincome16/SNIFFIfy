# SNIFFIfy

SNIFFIfy ist ein satirisches, mobiles Finger-Geschicklichkeitsspiel.

Die Spieler ziehen eine virtuelle Spur möglichst schnell und genau nach. Anschließend erzeugt die App eine fiktive Auswertung mit Zeit, Genauigkeit, Geschwindigkeit, Punkten, Rang und einer frei erfundenen „virtuellen Menge“.

## Wichtiger Hinweis

Die App misst, dokumentiert oder bewertet keinen echten Substanzkonsum. Sämtliche Mengen und Ergebnisse sind simuliert und dienen ausschließlich der Unterhaltung.

## Dateien

- `index.html` – Einstiegspunkt der App
- `style.css` – vollständiges mobiles Design
- `app.js` – Spielablauf, Auswertung, Speicherung und Bestenliste

## Lokal testen

Die `index.html` kann direkt im Browser geöffnet werden.

Für zuverlässiges Verhalten auf Mobilgeräten empfiehlt sich GitHub Pages.

## GitHub Pages aktivieren

1. Repository auf GitHub öffnen.
2. `Settings` öffnen.
3. `Pages` auswählen.
4. Unter `Build and deployment` die Quelle `Deploy from a branch` wählen.
5. Branch `main` auswählen.
6. Ordner `/ (root)` auswählen.
7. Auf `Save` drücken.

Die App sollte danach unter folgender Adresse erreichbar sein:

https://aiincome16.github.io/SNIFFIfy/
